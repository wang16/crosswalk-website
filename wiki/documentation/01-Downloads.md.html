The Crosswalk project provides binaries for multiple operating systems and platforms. See the links provided below for detailed per-operating system installation instructions.

Beta Release

Crosswalk 1.29.4.1 - Beta Build

NOTE: This is a Beta stabilization build of Crosswalk. The Beta channel is where Crosswalk is stabilized for an eventual Stable release. Only bug fixes go into the Beta channel; new feature development occurs in the minimally tested Canary releases.

See Release Methodology for a description of the Crosswalk release cycle and channel descriptions.

  • Selected Tizen 2.x Web APIs implemented: Bluetooth, Notification, Power, Time, System Settings, System Information, Download. See Tizen Extensions Crosswalk APIs.
